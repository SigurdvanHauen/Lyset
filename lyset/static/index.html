EV Charger — Register Scanner

Scans the charger's Modbus registers to discover the register map. Each range gets a fresh TCP connection. Also probes common ports (502, 6607, 8080…) so we can see what's actually listening.

Try FC3 first. If all ranges error, try FC4. Also try Slave ID 0, 100, or 200.

Device Discovery — FC43 + FC3 Sweep

Two-phase scan: FC43 (Read Device Identification) lists every device the SDongle knows about, then FC3 sweeps register ranges 0–3000, 30000–30524, 32000–47100. Slave ID order matches the Huawei HA integration: 0, 100, then 1–16. If scanning the inverter host, polling is paused and restarted automatically.
